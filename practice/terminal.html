Inro:

Vim is an incredible tool for editing text that is already installed on your computer!
In this website, I will be showing you the basics of Vim to get you started. Be warned, I will be somewhat accurate and even less thorough!
To start out, lets make a file called vimpractice.html. the command for that is
touch vimpractice.html
Now, to get into vim, we'll run
vim vimpractice.html
(we could've just run vim vimpractice.html from the beginning and it would've created it for us but this is just practice)

How Vim Works

Now you're in Vim! Right now you'll see a blank document. right now you're in command mode, so let's press i to get into insert mode. you should see -- INSERT -- in the bottom left of your screen. in insert mode you can move around with the arrow keys (not with your cursor; this was invented before cursors), and type letters into your document. Now press esc to go back into command mode. here are the most useful comands:

x
removes the letter you're currently on
dd
removes the line you're currently on
